# 2.x and 3.x

### 2.x

```php
require_once("/path/to/package/Mobile_Detect.php");

$detect = new Mobile_Detect();
$detect->isMobile();
$detect->isTablet();
```

### 3.x

```php
require_once "/path/to/package/src/MobileDetect.php"

$detect = new \Detection\MobileDetect;
$detect->isMobile();
$detect->isTablet();
```
